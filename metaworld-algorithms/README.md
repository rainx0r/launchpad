# metaworld-algorithms

A base Dockerfile containing all the required dependencies for running multi-task and meta-reinforcement learning algorithms in JAX on Metaworld.

Includes:

- Python@3.12
- CUDA & cuDNN for GPU acceleration.
- (TEMPORARY) Custom versions of Metaworld and Metaworld-Algorithms.
- JAX, Flax and Orbax.
- wandb for logging.
